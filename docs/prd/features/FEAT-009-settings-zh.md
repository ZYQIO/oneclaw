## 用户故事

**作为** OneClawShadow 的用户，
**我希望** 自定义应用的外观（主题和字体大小），
**以便** 应用符合我的视觉偏好，使用起来更加舒适。

### 典型场景

1. 用户偏好深色模式。用户打开设置，进入外观，将主题切换为 Dark。应用立即切换为深色配色。
2. 用户在系统中设置了较大的字体以满足无障碍需求。应用自动遵循该设置，以较大字号渲染所有文字，无需额外配置。

## 功能描述

### 概述

设置页面是应用配置的中心入口。目前已有"管理供应商"和"管理智能体"条目。本功能新增"外观"分区用于主题控制。字体大小跟随 Android 系统设置（V1 不提供应用内覆盖）。V1 仅支持英文界面。

### 设置页面布局

设置页面按分区组织：

```
Settings

Appearance
  Theme                          System default >

Providers & Models
  Manage Providers               Add API keys, configure models >

Agents
  Manage Agents                  Create and configure agents >

Usage
  Usage Statistics               View token usage by model >

Data & Backup
  Google Drive Sync              Not connected >
  Export Backup                  Save all data to a file >
  Import Backup                  Restore from a backup file >
```

### 外观

#### 主题
三个选项：
- **System default**：跟随 Android 系统深色/浅色模式设置。
- **Light**：始终使用浅色模式。
- **Dark**：始终使用深色模式。

选择结果持久化存储在本地设置中。切换后立即生效（无需重启应用）。

#### 字体大小
V1：跟随 Android 系统字体大小设置。应用对所有文字使用 `sp` 单位，会随系统字体大小自动缩放。V1 不提供应用内字体大小控制。

#### 语言
V1：仅支持英文。应用的 UI 字符串均为英文。其他语言（包括中文）的本地化推迟到未来版本，届时需要：
- Android 字符串资源（`strings.xml`）的翻译文件
- 内置智能体的本地化系统提示词（可选 -- 可保持英文）

## 验收标准

必须通过（全部必需）：
- [ ] 设置页面显示"外观"分区及"主题"条目
- [ ] 主题选项：System default、Light、Dark

- [ ] 主题偏好在应用重启后持久保留
- [ ] 应用文字随 Android 系统字体大小设置缩放
- [ ] 设置页面显示所有分区：Appearance、Providers & Models、Agents、Usage、Data & Backup

可选（锦上添花）：

- [ ] 应用内字体大小滑块（推迟到未来版本）

## UI/UX 要求

### 主题选择

点击"Theme"打开对话框或跳转至选择页面：

```
Theme

  ( ) System default
  (o) Light
  ( ) Dark
```

单选按钮选择。关闭对话框后立即应用所选项。

### 设置页面结构

每个分区有标题标签（`labelLarge`，`primary` 颜色），下方为列表项：
- 每个条目有标题和副标题/描述
- 点击后跳转至对应页面或打开对话框
- 与现有"Manage Providers"和"Manage Agents"条目保持一致

## 功能边界

### 包含
- 主题选择（System / Light / Dark）
- 跟随系统字体大小（通过 `sp` 单位自动实现）
- 设置页面按分区重新组织

### 不包含
- 应用内字体大小控制
- 语言选择 / 本地化
- 通知偏好开关
- 数据管理（清除缓存、管理存储）-- 推迟
- 关于 / 版本信息页面 -- 推迟
- 聊天显示设置（气泡样式、紧凑模式）-- 已在 FEAT-001 中涵盖

## 业务规则

1. 全新安装时主题默认为"System default"。
2. 主题切换立即生效 -- 无需重启应用。
3. 主题偏好存储在本地，并包含在同步/备份数据中（FEAT-007）。
4. 字体大小始终跟随 Android 系统设置。应用不覆盖系统字体大小。
